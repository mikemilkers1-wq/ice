# Outpost 31: Thin Ice

Static browser prototype for a Cluedo-like paranoia board game inspired by an arctic base survival mystery.

## Run

Open `index.html` in a browser, or upload the whole `thing-card-game` folder to GitHub Pages.

The folder must keep this shape:

```text
thing-card-game/
  index.html
  styles.css
  game.js
  assets/
```

If the title screen appears but no character cards appear, `game.js` is missing, blocked, or not in the same folder as `index.html`.

## Current Prototype Features

- Choose one of 12 station members, each with different stats, tools, perks, and response style.
- Move through 33 connected board nodes, including the base rooms and 12 outdoor points.
- Day 1 is a safe familiarization day.
- Day 2 begins with a dog infection event.
- Assimilation targets are randomized, not scripted.
- Ask characters about alibis, actions, suspicions, dog behavior, noises, blood traces, and unlocked clues.
- Public reputation, private friendship/enemy pressure, stress, breakdowns, frost exposure, night sleep, blood tests, accusations, and fire-based destruction are wired into the loop.
